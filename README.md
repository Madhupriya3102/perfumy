# Perfumy 

A responsive perfume storefront landing page built with **HTML, CSS, and JavaScript**. The site showcases a clean product layout with a fully responsive design that adapts from desktop to mobile, including a slide-in hamburger menu for smaller screens.

##  Live Demo

[View Live Site](https://madhupriya3102.github.io/perfumy/)

##  Features

- **Responsive design** — adapts smoothly across desktop, tablet, and mobile screens using CSS media queries
- **Mobile hamburger menu** — a slide-in side navigation menu for small screens, toggled with JavaScript
- **Product showcase** — clean, evenly-spaced product cards with images and descriptions
- **Interactive hover effects** — product images lift on hover for a dynamic feel
- **Search bar** — a styled search input in the header
- **About and Contact sections** — complete page structure with footer

##  Built With

- **HTML5** — page structure and semantic markup
- **CSS3** — styling, Flexbox layout, media queries for responsiveness, hover transitions
- **JavaScript** — DOM manipulation for the mobile menu toggle
- **Font Awesome** — icons for the hamburger menu and close button

##  Responsive Layout

- On screens **wider than 900px**: full horizontal navigation menu and side-by-side product cards
- On screens **900px and below**: navigation collapses into a hamburger menu that opens a slide-in side panel
